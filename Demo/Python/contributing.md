## 使用 GitHub Copilot 的提示模板

在撰寫新功能或修正時，請使用以下提示模板來生成程式碼：

""" 
任務：
- 撰寫純函式 <函式名稱>：輸入/輸出要清楚定義。
設計約束：
- 不修改外部狀態（pure function）
- 加入完整 docstring（參數、回傳、例外）
- 使用 typing 註記型別
品質要求：
- 若輸入含非預期型別，拋出 ValueError("Invalid input")
- 通過 pytest 測試（我已提供測試檔）
- 通過 flake8（PEP8）與 black 格式化
交付物：
- 函式實作 + 範例用法（若需要） 
"""
```
