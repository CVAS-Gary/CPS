## 快速開始

### 1. 啟動應用

在命令行中執行：

```bash
python app/main.py
```

### 2. 打開瀏覽器

在你的網頁瀏覽器中訪問：

```
http://localhost:5000
```

### 3. 開始計算

- 輸入一個數字或範圍
- 點擊「計算」按鈕
- 查看結果！

## 主要功能

### 功能 1：單一數字計算

**適用於**: 計算一個特定數字的 FizzBuzz 結果

**步驟**:
1. 在「單一數字計算」區域找到輸入框
2. 輸入你想計算的數字（例如 15）
3. 點擊「計算」按鈕或按 Enter
4. 結果立即顯示

**結果類型**:
- `Fizz` - 數字能被 3 整除
- `Buzz` - 數字能被 5 整除
- `FizzBuzz` - 數字能被 15 整除（3 和 5 都能整除）
- `數字` - 其他情況直接返回該數字

**範例**:
- 輸入 3 → 結果: `Fizz`
- 輸入 5 → 結果: `Buzz`
- 輸入 15 → 結果: `FizzBuzz`
- 輸入 7 → 結果: `7`

### 功能 2：範圍計算

**適用於**: 一次計算多個連續數字的 FizzBuzz 結果

**步驟**:
1. 在「範圍計算」區域輸入開始數字
2. 輸入結束數字
3. 點擊「計算」按鈕或按 Enter
4. 結果以網格形式顯示，不同類型用不同顏色標記

**顏色標記**:
- 🔴 紅色 - `Fizz`
- 🔵 藍色 - `Buzz`
- 🟣 紫色 - `FizzBuzz`
- ⚪ 白色 - 普通數字

**限制**:
- 最多一次可計算 10,000 個數字
- 最小數字是 1

**範例**:
- 開始: 1，結束: 15
- 結果: 1, 2, **Fizz**, 4, **Buzz**, **Fizz**, 7, 8, **Fizz**, **Buzz**, 11, **Fizz**, 13, 14, **FizzBuzz**

### 功能 3：統計信息

**顯示內容**:
- **Fizz 出現次數** - Fizz 在計算結果中出現了多少次
- **Buzz 出現次數** - Buzz 在計算結果中出現了多少次
- **FizzBuzz 出現次數** - FizzBuzz 在計算結果中出現了多少次

**何時更新**:
- 在你完成計算後自動更新
- 如果計算了單一數字，統計只顯示該數字的結果
- 如果計算了範圍，統計顯示整個範圍的統計

**應用場景**:
- 瞭解 FizzBuzz 的分佈
- 驗證計算結果
- 進行數據分析

## 逐步教學

### 教學 1：計算一個 FizzBuzz 數字

**目標**: 找出 30 是否是 FizzBuzz 數

**步驟**:

1. 在瀏覽器中打開 http://localhost:5000

   ![FizzBuzz 網站主頁](./images/homepage.png)

2. 在「單一數字計算」區域找到輸入框

3. 輸入數字 `30`

   ![輸入 30](./images/input-30.png)

4. 點擊「計算」按鈕

5. 看到結果 `FizzBuzz`

   ![結果](./images/result-fizzbuzz.png)

**解釋**: 30 能被 3 整除（30 ÷ 3 = 10），也能被 5 整除（30 ÷ 5 = 6），所以結果是 FizzBuzz。

---

### 教學 2：計算一個範圍

**目標**: 查看 1 到 20 的 FizzBuzz 結果

**步驟**:

1. 打開 http://localhost:5000

2. 向下滾動到「範圍計算」區域

3. 在「開始」輸入框輸入 `1`

4. 在「結束」輸入框輸入 `20`

   ![輸入範圍](./images/input-range.png)

5. 點擊「計算」按鈕

6. 看到結果網格：

   ```
   1  2  Fizz  4  Buzz  Fizz  7  8  Fizz  Buzz
   11 Fizz 13  14 FizzBuzz 16  17 Fizz  19  Buzz
   ```

7. 向下查看統計信息：
   - Fizz 出現次數: 6
   - Buzz 出現次數: 3
   - FizzBuzz 出現次數: 1

**解釋**: 
- Fizz (能被 3 整除): 3, 6, 9, 12, 15, 18 = 6 次
- Buzz (能被 5 整除): 5, 10, 15, 20 = 4 次（但 15 被計為 FizzBuzz，所以只有 3 次）
- FizzBuzz (能被 15 整除): 15 = 1 次

---

### 教學 3：使用統計信息驗證結果

**目標**: 確認我的計算是否正確

**步驟**:

1. 計算一個範圍（例如 1 到 100）

2. 查看統計信息

3. 驗證結果：
   - 1-100 中，被 3 整除的數有 33 個
   - 被 5 整除的數有 20 個
   - 被 15 整除的數有 6 個
   - Fizz 應該是 33 - 6 = 27 次
   - Buzz 應該是 20 - 6 = 14 次
   - FizzBuzz 應該是 6 次

## 提示和技巧

### 💡 提示 1：使用 Enter 鍵快速計算

不一定要點擊「計算」按鈕，你可以：
- 輸入數字後按 **Enter** 鍵
- 在結束數字輸入框按 **Enter** 鍵

### 💡 提示 2：測試大數字

試試看計算一個很大的數字，例如 999,999。FizzBuzz 對任何大小的數字都有效！

### 💡 提示 3：找出 FizzBuzz 數字的規律

在 1-100 範圍內，FizzBuzz 出現 6 次：15, 30, 45, 60, 75, 90

所有 FizzBuzz 數字都是 15 的倍數！

### 💡 提示 4：計算不同的範圍進行比較

- 1-50 的統計
- 51-100 的統計
- 看看有什麼不同

### 💡 提示 5：使用 API

如果你是開發者，可以使用後端 API 進行程序化調用：

```bash
curl "http://localhost:5000/api/fizzbuzz?number=15"
```

詳細信息見 [API_DOCUMENTATION.md](./API_DOCUMENTATION.md)。

## 故障排除

### 問題 1：應用無法啟動

**症狀**: 運行 `python app/main.py` 後出現錯誤

**解決方案**:

1. 確認你已安裝依賴：
   ```bash
   pip install -r requirements.txt
   ```

2. 確認 Python 版本是 3.10 或更高：
   ```bash
   python --version
   ```

3. 確認虛擬環境已激活

4. 查看詳細錯誤信息並搜索解決方案

### 問題 2：無法訪問網站

**症狀**: 輸入 http://localhost:5000 後頁面無法加載

**解決方案**:

1. 確認應用正在運行（見終端輸出）

2. 檢查是否是正確的地址：`http://localhost:5000`（注意不是 https）

3. 嘗試其他端口（如果 5000 被佔用）

4. 檢查防火牆設置

### 問題 3：計算結果看起來不對

**症狀**: 結果與預期不符

**檢查清單**:

1. ✅ 確認輸入是整數
2. ✅ 確認沒有多餘的空格
3. ✅ 對於範圍，確認開始數字 ≤ 結束數字
4. ✅ 重新加載頁面並重試
5. ✅ 清除瀏覽器緩存

### 問題 4：輸入框不接受我的輸入

**症狀**: 無法在輸入框中輸入

**解決方案**:

1. 確認你在點擊輸入框後才開始輸入
2. 檢查輸入框是否突出顯示（淺藍色邊框）
3. 嘗試刷新頁面
4. 嘗試使用不同的瀏覽器

### 問題 5：頁面加載緩慢

**症狀**: 計算大範圍時頁面反應遲緩

**説明**: 這是正常的。計算 10,000 個數字需要時間。

**改進方式**:

1. 先計算較小的範圍
2. 等待計算完成後再進行下一步操作
